Frequently Asked Questions

    Tell me a bit about this project
  • If you ask anyone who has spent any amout of time trying to navigate the national park webiste to book campsites and they will tell you it's generally a very confusuing process. My goal with this app was to create a simple and easy to use interface to allow users to quickly and easily find and reserve a campsite.
    What technoly did you use.
  • The Axios library was used to make requests from the front-end.
  • Using express, cors and sequelize, I created a small database and then manipulated that data on the back end using SQL requests
  • JS was implented to create dynamic responses for user input and to return created elements
  • Lastly, HTML was used to create the strutuce and CSS implemented for basic styling
    Why did you choose this tech stack?
  • As my first major project on my journey to becoming a web developer, I wanted to showcase my new knowledge of JS to ensure that the website was dynamic
    What did you learn from building this project?
  • This foundational project was my first opportunity to "put it all together". It was a great opportunit to reinforce my new skills and to showcase what I can build using a full stack.
